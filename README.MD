## Market Basket Analysis API
Этот проект представляет собой микросервис на базе FastAPI, который использует алгоритм Apriori для анализа рыночной корзины. Он выявляет ассоциативные правила между товарами в транзакциях (например, «покупатели, которые берут товар А, часто берут и товар Б»).

### Установка
Клонируйте репозиторий:
```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
Установите необходимые зависимости:

```pip install -r requirements.txt```

Убедитесь, что в корневой директории находится файл с данными Transactions.csv (разделитель ;).

### Запуск приложения
Запуск сервера разработки осуществляется с помощью uvicorn. Выполните следующую команду в терминале:

```uvicorn main:app --reload```

- main - это имя вашего Python-файла (без .py).
- app - это экземпляр FastAPI внутри этого файла.
- --reload - флаг для автоматической перезагрузки сервера при изменении кода (полезно при разработке).

После запуска сервер будет доступен по адресу: http://127.0.0.1:8000

### API Endpoints
1. Получение ассоциативных правил
URL: _/pairs_

- Метод: GET

- Описание: Возвращает список найденных комбинаций товаров (пар/троек) с высоким уровнем поддержки и достоверности.

2. Документация
Swagger UI: _/docs_ - интерактивная документация для тестирования API.


### Структура данных
Программа ожидает CSV-файл, где:

- Колонка number содержит идентификатор транзакции.
- Остальные колонки - названия товаров, где 1 означает покупку, а 0 - её отсутствие.
